Co je API?
==========

Když si jako laik na Wikipedii najdete `článek o API <https://cs.wikipedia.org/wiki/API>`__, nejspíš z něj moudří nebudete. Tyto materiály, nazvané `Co je API? <http://cojeapi.cz>`__, se snaží API vysvětlit běžným lidem a zároveň umožnit začátečníkům v programování, aby se s API naučili pracovat.
